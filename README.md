知乎日报·今日
==========

### 简介

一款轻量级的知乎日报，能够抓取知乎日报上的所有内容，能够缓存部分内容，不支持离线查看。开发环境为Eclipse。

### 依赖

android-support-v7-appcompat.jar  
handmark pulltorefresh  
nostra13 universalimageloader
